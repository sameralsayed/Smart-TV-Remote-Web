# Smart TV Remote Web

📺 A sleek and fully functional universal Smart TV Remote web app with Roku-style controls, screen mirroring, casting, and channel launcher.

![Smart TV Remote Preview](https://via.placeholder.com/800x400/000000/ffffff?text=Smart+TV+Remote)

## ✨ Features
- 🔘 **Full Remote Controls** — Power, Volume, Mute, D-Pad, Playback
- ⌨️ **On-screen Keyboard** — Type directly on your TV
- 📺 **Quick Channels** — Launch Netflix, YouTube, Disney+, etc.
- 📱 **Screen Mirroring** — Simulate mirroring your phone to TV
- 📤 **Content Casting** — Cast photos/videos to TV
- 📲 **Touch-friendly Design** — Large buttons optimized for mobile
- 🌐 **Fully Responsive** — Works perfectly on phones, tablets, and desktops

## 🚀 How to Use
1. Open `index.html` in any modern browser
2. Use the power/volume/D-pad buttons
3. Tap channels to "launch" them
4. Try Screen Mirroring and Cast features
5. Open the keyboard to type on TV

**Note**: This is a realistic simulation demo. No actual TV is controlled.
